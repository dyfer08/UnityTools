# UnityTools
A series of useful Unity tools


## HCM (Hierarchy color manager)
Assign colors to the gameobjects in your hierarchy window. This is based on tags.

![HCM in Unity](https://ferdinanddervieux.com/ImageHosting/HCM.png)
